Welcome to Radiomics Features Extraction for TCIA ISPY2 Images

This workflow was created as part of Henry Fessler's final project for CMSE-890-602 Reproducible Computational Workflows.

Workflow Summary

This workflow takes the identifying information for a patient and imaging event from the ISPY2 Image Collection and provides the radiomics for that imaging event.

For a more complete view of the workflow please see the psuedocode or dataflow diagram.

To run the workflow please see the quickstart guide. To check your installation, please see the testing guide. For more details on the package requirements, please see the installation details.
